- **手机** (< 768px) - 堆叠布局，侧边菜单

## 🚀 下一步操作

### 1. 安装运行环境
- 安装 Node.js (v18+)
- 在项目目录运行 `npm install`
- 运行 `npm run dev` 启动开发服务器

### 2. 自定义内容
- 修改课题组名称和介绍
- 更新团队成员信息
- 添加真实的研究项目和出版物
- 更新联系信息

### 3. 部署上线
- 推送代码到 GitHub
- 使用 Vercel 或 Netlify 一键部署

## ⚠️ 注意事项

1. **依赖安装** - 需要先安装 Node.js 和项目依赖
